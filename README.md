# NYC Public Schools SAT Performance Analysis

## Overview
This project analyzes SAT performance across New York City public schools to identify top-performing schools, examine score variations across boroughs, and understand the distribution of academic achievement.

## Dataset
The analysis uses `schools.csv` containing SAT scores (math, reading, writing) for NYC public schools across all five boroughs.

## Key Findings
- **Top School**: Stuyvesant High School leads with highest math score (754) and total SAT (2144)
- **Borough Leader**: Staten Island has the highest average SAT score
- **Score Correlation**: Schools strong in math also excel in reading
- **Distribution**: Most schools score 1100-1300, with few elite schools above 1800
